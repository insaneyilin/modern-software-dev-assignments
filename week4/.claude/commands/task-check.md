# Task Completion Check

Check progress on tasks from `docs/TASKS.md`.

**Usage**: `/task-check [task_number]`

**Steps**:
1. Read `docs/TASKS.md`
2. If task_number provided: Check specific task
3. Otherwise: Check all tasks
4. Verify implementation status:
   - Check if endpoints exist
   - Check if tests exist
   - Check if frontend updated (if applicable)
5. Output status report with missing pieces
